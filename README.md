### :new_moon_with_face: before installing pyxel
**Windows**
` pip install -U pyxel `

**Mac**
`brew install python3 sdl2 sdl2_image`
or
`pip3 install -U pyxel`

**Linux**
Ubuntu: `sudo -H pip3 install -U pyxel`
